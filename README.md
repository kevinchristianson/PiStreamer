# PiStreamer
This repo contains a script for setting up a Raspberry Pi as a Plexamp Streamer, including configuring an external touch-display kiosk.

## Features:
- Easy setup from a fresh flash of Raspberry Pi OS Lite
- External display kiosk setup
  - Display auto-sleep during inactivity
  - Display keep-awake during music playback
  - Display wake on music playback
